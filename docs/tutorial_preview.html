Tutorial

First camera preview.

Initialize a camera and open a live preview using the MEYELens Python API.

Goal

This tutorial checks that the package is installed correctly and that your camera can be opened from Python.

Python 
from meyelens import meye, camera

cam = camera.Camera(camera_index=0)
meye = meye.Meye()

meye.preview(cam)
Tip: If the wrong camera opens, try changing camera_index=0 to camera_index=1 or camera_index=2.
