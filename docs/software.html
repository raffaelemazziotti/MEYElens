Installation

We recommend installing MEYELens in a dedicated Python environment to avoid dependency conflicts.

Recommended install

Create and activate a dedicated Conda environment, then install MEYELens with the standard TensorFlow dependency set.

Command 
conda create -n meyelens python=3.10 -y
conda activate meyelens
pip install meyelens[tf]
Tip: This installs MEYELens together with a standard TensorFlow dependency set. Depending on your system, TensorFlow may run on CPU unless GPU support is configured separately.

TensorFlow GPU support (optional)

If you want GPU acceleration, configure TensorFlow first by following the official TensorFlow installation guide for your platform. After verifying that TensorFlow can access the GPU, install MEYELens without the TensorFlow extra.

Warning: GPU enablement depends on your operating system, CUDA/cuDNN compatibility, and TensorFlow version. Follow the official TensorFlow installation guide exactly for your platform before installing MEYELens.
Command 
pip install meyelens
Tip: This keeps your manually configured TensorFlow installation intact and avoids replacing it with the default dependency set from MEYELens[tf].

Headless package

If you want only the core Python package without the PyQt offline GUI, install the separate headless distribution.

Command 
pip install meyelens-headless

To install the headless package with TensorFlow dependencies:

Command 
pip install "meyelens-headless[tf]"
Warning: Do not install meyelens and meyelens-headless into the same environment. Use separate environments if you need to test both distributions.
